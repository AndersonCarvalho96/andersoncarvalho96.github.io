## Descrição
Desenvolvimento de um dashboard interativo focado na análise profunda de transações financeiras. [cite_start]O projeto transforma dados brutos em indicadores estratégicos para suporte à tomada de decisão contábil e operacional. [cite: 55]

## Funcionalidades Principais
- [cite_start]**Análise de KPIs:** Visualização em tempo real de receita, despesas, lucro líquido e fluxo de caixa. [cite: 57]
- [cite_start]**Métricas de Performance:** Cálculo automático de margem de lucro, eficiência operacional e qualidade das transações. [cite: 57]
- [cite_start]**Modelagem Relacional:** Estrutura de dados baseada em MER (Modelo Entidade-Relacionamento) e DER (Diagrama Entidade-Relacionamento). [cite: 56]
- [cite_start]**Normalização de Dados:** Processamento de dados para garantir consistência em consultas SQL complexas. [cite: 56]

## Tecnologias Utilizadas
- [cite_start]**Python (Pandas & Streamlit):** Motor de processamento de dados e interface do usuário. [cite: 55]
- [cite_start]**SQLite & SQL:** Armazenamento robusto e consultas estruturadas para recuperação de indicadores. [cite: 56]
