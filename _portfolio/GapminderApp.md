## Descrição
Este projeto é um explorador de dados globais inspirado na ferramenta Gapminder. [cite_start]Ele permite visualizar a evolução socioeconômica de diferentes países ao longo das décadas através de gráficos dinâmicos e interativos. [cite: 60]

## Funcionalidades Principais
- [cite_start]**Análise Multidimensional:** Cruzamento de dados de expectativa de vida, PIB per capita e crescimento populacional. [cite: 60]
- [cite_start]**Otimização de Performance:** Adaptado para processamento via CSV local, garantindo maior estabilidade e eliminando dependências de limites de nuvens externas (como a versão gratuita do Databricks). [cite: 61]
- **Visualização Dinâmica:** Uso de gráficos de dispersão animados que mostram a mudança dos dados ao longo do tempo.

## Tecnologias Utilizadas
- [cite_start]**Python (Pandas):** Manipulação e limpeza de grandes conjuntos de dados (Datasets). [cite: 60]
- [cite_start]**Plotly:** Criação de gráficos interativos e responsivos. [cite: 60]
- [cite_start]**Streamlit:** Framework para publicação rápida da aplicação web. [cite: 60]
