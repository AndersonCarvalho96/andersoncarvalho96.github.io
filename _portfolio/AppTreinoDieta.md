## Descrição
O **Olympia Shape** é um aplicativo mobile desenvolvido em **Flutter**, voltado para o **gerenciamento de treinos e dietas**.  
Ele permite que o usuário organize e acompanhe seus exercícios físicos e alimentação diária de forma **simples, prática e intuitiva**, funcionando totalmente de forma **offline**.

---
## Funcionalidades Principais
**🏋️ Treinos**  
- Criação, edição e exclusão de treinos personalizados.  
- Registro de séries, repetições, cargas e observações.  
- Organização dos exercícios por grupo muscular (Peito, Costas, Pernas, etc.).  
- Treinos pré-determinados disponíveis a partir da versão 2.1.  

**🍎 Dietas**  
- Adição, edição e exclusão de refeições.  
- Campos personalizados para nome, horário e observações.  
- Interface simples e direta para o uso diário.  

**⚙️ Outras melhorias**  
- Estrutura modular e organizada.  
- Layout responsivo e melhorias visuais.  
- Campo de repetições configurado para aceitar intervalos (“8-10”).  
- Sistema de confirmação antes de exclusões.  

---
## Tecnologias Utilizadas
**Flutter** – Framework multiplataforma utilizado no desenvolvimento do app.  
**Dart** – Linguagem de programação base do projeto.  
**Armazenamento local** – Estrutura interna para salvar dados sem depender de banco de dados online.  

---
## Como Funciona
O aplicativo utiliza **armazenamento local** para registrar as informações de treinos e dietas, garantindo funcionamento mesmo sem internet.  
A interface foi projetada para ser **intuitiva e acessível**, permitindo ao usuário adicionar, editar e visualizar treinos e refeições de forma rápida e organizada.  
O código segue uma arquitetura modular, facilitando manutenção e futuras expansões.
